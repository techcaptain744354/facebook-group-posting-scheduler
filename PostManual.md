# Facebook Group Posting Scheduler Documentation

## Overview

MadHub's powerful automation features leverage advanced Android functionalities to enable seamless **Facebook Group Auto-Posting**. This feature empowers users to schedule and manage posts across multiple Facebook groups effortlessly, enhancing content outreach and engagement.

## Feature Capabilities

### Facebook Group Auto-Posting

MadHub's **Facebook Group Auto-Posting** allows users to automate the process of posting content in various Facebook groups. This feature encompasses several robust capabilities:

- **Content Rotation**: Automatically rotates content among different posts to prevent redundancy and maintain audience interest.
- **Flexible Input Modes**: Supports both manual input and file uploads, allowing users to schedule posts from a text file for bulk operations.
- **Custom Configurations**: Users can specify the number of posts per group and set limits on total postings, granting full control over the posting frequency.
- **Image Insertion Support**: Users can enhance their posts by including images, making the content more visually appealing.
- **Loop Mode**: Allows for repeated posting of selected content at defined intervals, which is ideal for campaigns requiring consistent visibility.

## Technical Usage

To effectively utilize the **Facebook Group Auto-Posting** feature, follow these instructions:

1. **Set Up Your Facebook Groups**: Ensure you have joined the groups where you wish to post.
2. **Access the Auto-Posting Tool**: Navigate to the MadHub interface and select **Facebook Group Auto-Posting** from the features menu.
3. **Input Your Content**:
   - **Input Mode**: Type your content directly into the text box for quick posts.
   - **File Mode**: Upload a text file containing multiple posts to execute bulk scheduling.
4. **Configure Posting Parameters**:
   - **Set Post Count**: Define how many posts you want to publish in a single group and total across all groups.
   - **Image Upload**: Attach images as necessary to enhance the posts.
5. **Choose Posting Frequency**: Set the intervals between posts to optimize engagement without overwhelming group members.

### Example Command
```plaintext
madHub.postToGroups("Post Title", "This is the body of the post", "path/to/image.jpg", 5, 10);
```
In this example:
- `Post Title` is the title of the post.
- `"This is the body of the post"` is the content of your post.
- `"path/to/image.jpg"` is the image file you want to include.
- `5` represents the number of times to post in a single group.
- `10` represents the total number of groups to target.
